# 🧟 HORDE — Zombie Apocalypse

**HORDE** — выживание в мире, захваченном ордами мёртвых. Minecraft 1.20.1 (Forge), кастомный сервер с огнестрелом, сотней видов зомби, заброшенными постройками, базами и голосовым чатом.

## Как играть (для игрока)
1. Скачай **`HORDE.exe`** (≈160 МБ).
2. Запусти → введи **ник** → нажми **ИГРАТЬ**.
3. Лаунчер сам поставит Java, Minecraft, Forge и все моды — больше ничего не нужно.
4. На сервере: `/register пароль пароль`, потом `/login пароль`. Через ~12 сек получишь стартовый кит + книгу-гайд с правилами.

Есть лицензия? Жми **«У меня лицензия»** — играешь своим аккаунтом через официальный лаунчер.

## Что внутри
- 🔫 **Оружие** — TaCZ + пак Suffuse: пистолеты, винтовки, дробовики, пулемёты, РПГ. Собираются на Оружейном столе.
- 🧟 **100+ видов зомби** (Infectious): орды, мутанты, военные, учёные — атакуют ночью.
- 🏚 **Постройки** — заброшенные дома, военные базы, бункеры, больницы/руины, техника (грузовики, танки, вертолёты) генерируются по миру.
- 🛡 **Базы** — приват своей территории (Open Parties and Claims), чтобы не грифили.
- 🎒 **Рюкзаки** (Sophisticated Backpacks), 🎙 **голосовой чат** (proximity), 🌫 **шейдер Complementary** + хоррор-текстуры.
- 🔐 **Регистрация** (LoginSystem) — ник защищён паролем.
- 🚫 Взрывы не ломают постройки (кроме криперов), TNT отключён.

## Правила
1. Не грифь чужое — привать базу.
2. Без читов и багоюза.
3. Не оскорбляй игроков.
4. Лут и пушки ищи в сундуках построек.
5. Ночью — орды. Береги базу.
